# DESIGN.md — 昆明学院国防先锋队官网

## 产品定位

一群怀揣军旅梦想的大学生，凌晨出操、烈日拉练、守护国旗。**官网不是商业 SaaS 的展示页，而是营区大门口那块"国防先锋队"牌匾的延伸**——把仪式感、纪律性、集体荣誉感翻译成网页语言。

**校验第一直觉**：第一反应是"军绿+金色+圆角卡片"——这等于把军装画成了儿童贴纸。我们需要**硬切角、铜质金属、纸张肌理、罗马数字、军衔臂章的斜条纹**，让它有"旧军牌+新青年"的双重温度。

## 气质与意象

**关键词**：军牌铜质光泽、晨雾中的营房、勋章红绒底座、军绿帆布、军衔斜条、罗马数字、登山绳索、金属刻字、营火余温。

**具象场景**：
- 凌晨6点的营房门口，钢盔擦得锃亮，远处薄雾里有一面红旗
- 老式军牌挂在黑色帆布上，金属边磨损出暗金色
- 营区训练场的水泥墙上，用白漆写着"团结 紧张 严肃 活泼"
- 收藏柜里的勋章，红色绒布底座上暗金色徽章折射光线

## 视觉策略

- **不做传统圆角卡片**，改用 **`clip-path` 切角**（军牌/臂章造型）做硬朗轮廓
- **斜条纹**作为分区装饰（致敬军衔V字章）
- **纸张肌理叠加**（细密网格 SVG 噪点）增加历史厚重感
- **暗金铜质**通过渐变 `linear-gradient(135deg, #d4af37, #b8941f, #d4af37)` 模拟金属反射
- **数字徽章 + 罗马数字**作为视觉点缀（如 "EST. 2018"、"II. 荣誉"）
- **图标**：Bootstrap Icons（bi-*），不混入 emoji

## 配色方案

| 用途 | 颜色 | 意象 |
|------|------|------|
| 主色 | 军绿 `#2c5e2e` | 65式军装绿 |
| 主色深 | 深绿 `#1a3d1d` | 暗色模式 / 阴影色 |
| 强调色 | 勋章金 `#d4af37` | 老式铜质勋章 |
| 强调深 | 旧金 `#b8941f` | 暗金阴影 |
| 背景亮 | 米白 `#f5f1e8` | 老营房纸张 |
| 背景暗 | 炭墨 `#0d100d` | 暗色模式 / 营区夜空 |
| 文字亮 | 墨黑 `#1a1a1a` | 正文 |
| 文字暗 | 米白 `#f0ead6` | 暗色模式文字 |
| 警示 | 锈红 `#8b3a2f` | 极少用，仅作状态点 |

**配色口诀**："**墨绿是底，勋章金是睛，米白是纸，炭墨是夜**"。

## 字体排版

- **主标题**：`Oswald`（军事海报常用，紧凑有力）+ `Noto Serif SC` 中文（庄重衬线）
- **正文**：`Inter` + `Noto Sans SC`
- **数字徽章**：`Oswald` 配合 `letter-spacing: 0.15em`（金属刻字感）
- **小标签**：大写英文 + 0.2em 字间距（臂章风格）

**字体加载**：`fonts.googleapis.cn`（CN 区域优先）

## 动效与交互

- **入场动画**：滚动到视区时，从下方淡入并上移 20px（`translateY(20px) → 0`）
- **数字滚动**：统计区数字从 0 滚动到目标值，使用 `requestAnimationFrame` + 缓动函数
- **卡片悬停**：金属光泽扫过（`::before` 伪元素 + `transform: skewX(-20deg)` + `left: -150% → 150%`），不靠柔光阴影
- **暗色切换**：图标 180° 旋转 + 月亮🌙 ↔ 太阳☀️
- **按钮悬停**：背景从右向左填充（`background-size: 0% 100% → 100% 100%`）
- **导航滚动**：滚动超过 80px 时背景从透明渐变到深色（毛玻璃 + 阴影）

## 页面结构

| 区块 | 实现要点 |
|------|---------|
| 1. 固定导航栏 | 半透明 + `backdrop-filter: blur(10px)`，滚动后变实色 |
| 2. 全屏轮播 | 暗色遮罩 `filter: brightness(0.5)`，渐变文字层 |
| 3. 统计条 | 4个数字（成员数 / 训练人次 / 活动数 / 荣誉数）+ 罗马数字编号 |
| 4. 社团简介 | `.col-lg-6` 图文各半，左侧文字+徽章，右侧图（带切角） |
| 5. 活动预告 | `.col-md-4` 三列卡片，悬停金属光泽扫过 |
| 6. 成员介绍 | **Flex 一维布局**（`flex-wrap: wrap; justify-content: center; gap: 2.5rem`），圆形头像 |
| 7. 发展历程 | 垂直时间线（`::before` 伪元素做连接线），左右交替 |
| 8. 加入我们 | CTA 大区块，触发模态框表单 |
| 9. 页脚 | `.col-md-4` 三列 + 社交图标，铜质金属色 |
| 10. 返回顶部 | 固定右下角圆形按钮，滚动 300px 后出现 |

## 设计禁忌

- ❌ **不**用通用 SaaS 圆角软阴影卡片
- ❌ **不**用科技蓝紫色渐变
- ❌ **不**混入 emoji 表情
- ❌ **不**用玻璃拟态堆叠（仅导航栏克制使用）
- ❌ **不**做过度对称的几何极简（应有人文温度）
- ❌ **不**用卡通化插画
- ❌ **不**让文字直接贴在图片上无遮罩
- ❌ **不**用纯黑色背景（破坏军旅质感，用炭墨 `#0d100d`）
